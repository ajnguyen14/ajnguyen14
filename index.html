About

Bioengineering student transferring to UC Berkeley in fall 2026. Two years of wet lab research across vascular biology and Drosophila genetics at SJSU, self-taught in deep learning.

My current focus is on computational approaches to protein function prediction — specifically how transformer-based protein language models encode evolutionary and structural information that can be repurposed for functional annotation. Long-term, I'm interested in computational drug discovery and eventually a PhD.

I like working at the interface where biology gets weird enough that classical methods start to fail.

Projects

Protein Function Classifier V2 In Progress

Multitask deep learning model using ESM-2 transformer embeddings for simultaneous prediction of EC class and GO biological process terms. Benchmarked against CLEAN using homology-aware train/test splits via CD-HIT to prevent data leakage from sequence similarity.

github.com/ajnguyen14/protein-classifier ↗
Protein Function Classifier V1

First end-to-end ML pipeline for protein function prediction using scikit-learn and sequence-derived features. Built to understand the baseline before moving to embedding-based approaches.
